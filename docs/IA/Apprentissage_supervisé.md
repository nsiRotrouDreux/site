
# Apprentissage Supervisé en IA

## Définition

L'apprentissage supervisé est une méthode d'apprentissage automatique où un modèle est entraîné sur un ensemble de données étiquetées. Chaque exemple d'entraînement comprend une entrée et sa sortie attendue correspondante.

## Caractéristiques principales

- **Données étiquetées** : Les données d'entraînement incluent des labels ou des réponses correctes
- **Objectif clair** : Apprendre à prédire des outputs à partir d'inputs
- **Supervision** : Un superviseur (humain ou algorithme) valide l'apprentissage

## Types principaux

### Régression
Prédire une valeur continue (ex: prix d'une maison, température)

### Classification
Prédire une catégorie discrète (ex: spam/non-spam, chat/chien)

## Exemples d'algorithmes

- Régression linéaire
- Arbres de décision
- Réseaux de neurones
- Support Vector Machines (SVM)
- k-Nearest Neighbors (k-NN)

## Avantages et Limitations

**Avantages** : Haute précision, bien compris, efficace avec suffisamment de données

**Limitations** : Requiert beaucoup de données étiquetées, coûteux en annotation manuelle
